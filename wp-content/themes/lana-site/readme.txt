Bootstrap based Responsive WordPress Theme

== Description ==

Lana Site is a simple Bootstrap framework based WordPress theme.
It's a 12 grid responsive theme, optimized for tablets and phones.
This is the right theme for a blog or a simple website.

= Lana Site =
[Lana Site](http://wp.lanaprojekt.hu/blog/wordpress-themes/lana-site/)

== Installation ==

Manual installation:

1. Upload the theme files to the `/wp-content/themes/lana-site` directory

Installation using “Add New Theme”

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for ‘lana-site’
3. Click the ‘Install’ button to open the theme’s repository listing
4. Click the ‘Install’ button

Activiation and Use

1. Activate the Theme through the ‘Themes’ menu in WordPress
2. See Appearance -> Theme Options to change theme specific options
